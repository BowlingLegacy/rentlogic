Owner Dashboard Setup Wizard {% if user.is_superuser or user.role == "admin" %} Platform Revenue {% endif %}
Owner rent money Rent, deposits, utilities, and tenant payments should route to the owner or property Stripe destination when a connected account is active.
RentalReadyPro revenue Subscriptions, platform payment fees, application processing, background-check admin, setup, referrals, and advertising are tracked separately in Platform Revenue.
Fallback behavior If no owner/property Stripe route exists, the system falls back to the platform Stripe account. Use connected accounts before live owner rent collection.

Recommended for most owners

  • Use one owner Stripe connected account when one owner receives rent for multiple properties.
  • Add property overrides only when a property needs its own payout destination.
  • Do not paste Stripe secret keys here. This page stores routing IDs only.

Before going live

  • Confirm the route shows Ready for each property accepting online payments.
  • Run a small test payment before tenants are asked to use the portal.
  • Keep RentalReadyPro platform revenue separate from owner rent money.
